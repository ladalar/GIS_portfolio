# Images

This directory holds images used by the portfolio.

## switrs/
Place your timelapse animation frames here.
Name them sequentially, e.g.:

```
frame_01.png
frame_02.png
frame_03.png
...
```

Then register each path in the `FRAMES` array inside `js/main.js`.

## certificates/
Place your ArcGIS certificate images here, e.g.:

```
esri_arcgis_pro.jpg
esri_spatial_analyst.jpg
...
```

Then add a new `<div class="cert-card">` block in the `#certificates`
section of `index.html` for each certificate.
