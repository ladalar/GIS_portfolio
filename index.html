Project Overview

This project visualizes historical traffic accidents in California using the Statewide Integrated Traffic Records System (SWITRS) dataset. Using QGIS, thousands of point locations representing traffic collisions were processed and aggregated into monthly intervals to create a dynamic temporal heatmap. The visualization highlights patterns and trends in accident density over time, enabling insights into high-risk areas and seasonal fluctuations.

Technical Details

  • Data Source: California SWITRS (traffic collision records)
  • Tools & Technologies: QGIS, Temporal Controller, Heatmap renderer, GeoPackage, qgis2web
Key Steps
  1. Loaded and cleaned CSV collision data with geospatial coordinates.
  2. Converted the CSV into an editable memory / GeoPackage layer for analysis.
  3. Created a monthly aggregation field to support temporal animation.
  4. Applied a heatmap renderer to visualize accident density.
  5. Configured temporal properties in QGIS for smooth month-by-month animation.
  6. Exported an interactive web map using qgis2web for portfolio presentation.

Outcome & Insights

  • Dynamic visualization of accident trends over time.
  • Clear identification of hotspots and seasonal patterns.
  • Interactive map suitable for web presentation, providing both technical demonstration and analytical storytelling.

Project Overview

This project develops an interactive geospatial analysis to visualize and analyze rent price trends across Southern California using ZIP-level data. Zillow Observed Rent Index (ZORI) data was integrated with U.S. Census ZCTA shapefiles in QGIS to create a spatially accurate representation of housing costs over time. A dynamic visualization pipeline built with PyQGIS automates temporal rendering across 100+ monthly time steps, enabling a time-lapse animation that highlights rent growth patterns and regional disparities.

Technical Details

  • Data Sources: Zillow Observed Rent Index (ZORI) CSV; U.S. Census Bureau ZCTA shapefiles
  • Tools & Technologies: QGIS, PyQGIS, Python, Graduated Symbology, Spatial Join, GeoPackage
Key Steps
  1. Performed data preprocessing and spatial joins to align non-spatial ZORI CSV data with ZCTA geographic boundaries.
  2. Addressed ZIP code formatting inconsistencies and partial data coverage during cleaning.
  3. Filtered and extracted Southern California regions using spatial selection techniques.
  4. Applied graduated symbology to represent rent values with a consistent classification scheme across all frames.
  5. Scripted programmatic map updates in PyQGIS to iterate through 100+ monthly time steps automatically.
  6. Exported frame-by-frame PNG outputs and assembled them into a time-lapse video for storytelling.

Outcome & Insights

  • Comprehensive ZIP-level view of rent trends from 2015 to 2026, revealing sustained growth across the region.
  • Clear identification of high-cost clusters and emerging rent-growth corridors in Southern California.
  • Time-lapse animation provides an accessible, compelling narrative of regional housing cost dynamics over a decade.
